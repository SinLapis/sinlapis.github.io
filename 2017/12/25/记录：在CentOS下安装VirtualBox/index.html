记录：在CentOS下安装VirtualBox
2017-12-25|Linux
  • 安装kernel-devel，注意和系统内核版本对应。查看系统内核版本：

uname -r
  • 安装其他依赖：

yum install -y SDL gcc make perl
  • 到官网上下载CentOS对应的rpm安装包，下载并安装即可。
